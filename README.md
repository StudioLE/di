# Alnwick

## Introduction

A self-hosted podcast library that works as both a **web application** and a **command-line tool**.

### Features

- Self-hosted podcast library management
- Web-based player and interface
- Command-line tools for automation and scripting
- Built entirely in Rust for performance and reliability

## Web Application

Self-hosted web interface for adding podcasts and downloading podcasts.

Built with [Dioxus](https://dioxuslabs.com), a [Rust](https://www.rust-lang.org) framework for building fullstack web,
desktop, and mobile apps.

- [Install the web app](INSTALL-WEB.md)
- [Build the web app with Docker](BUILD-WEB-DOCKER.md)
- [Build the web app with cargo](BUILD-WEB.md)

## CLI

A CLI providing full control for for adding podcasts and downloading podcasts with advanced filtering.
